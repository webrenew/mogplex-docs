The web app is the shared control plane for Mogplex.

Use it to sync repositories, shape reusable agents, build GitHub-triggered
automations, inspect runs and model calls, and manage the settings the CLI can
reuse.

## Current surface

- **Projects** imports repos, organizes them into workspaces, and launches sandboxes.
- **Agents** manages reusable agent definitions plus shared skills, rules, and context.
- **Automations** is the flow-builder surface for GitHub-triggered work.
- **Observability** tracks runs, pressure, failures, and model/tool calls.
- **Settings** manages connections, provider keys, CLI tokens, models, and MCP servers.
