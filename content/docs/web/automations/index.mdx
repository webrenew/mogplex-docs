title: Automations
description: Run Mogplex on every pull request — automated GitHub code reviews and follow-up actions.
---

Automations connect a Mogplex project to a GitHub repository and run on
pull-request events. Use them to post AI code reviews, gate merges, or
trigger follow-up flows without leaving GitHub.

## What you can do

- Review every PR opened against a repo
- Comment inline with findings, suggestions, and risk callouts
- Re-run a review when new commits are pushed
- Chain into a Mogplex flow or agent for follow-up work
