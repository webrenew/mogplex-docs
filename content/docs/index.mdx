## Quickstart

Install the CLI and run your first command.

```bash
curl -fsSL https://install.mogplex.dev/cli.sh | sh
mogplex login
mogplex exec "summarize this repo"
```

Prefer the hosted experience? Sign in to the [App](/web) and connect a GitHub installation to get started.
