Mogplex ships in two surfaces.

Use the CLI when you want local, repo-in-place agent work from your terminal.
Use the web app when you want the shared control plane for projects, agents,
automations, triggers, observability, and settings.

## Quickstart

Install the CLI and start a first task. If you are not authenticated yet,
`mogplex` now walks you into browser sign-in or local API-key setup.

```bash
curl -fsSL https://www.mogplex.com/install.sh | sh
mogplex
```

If you prefer a direct provider key, `ANTHROPIC_API_KEY` and `OPENAI_API_KEY`
work as expected too.

Prefer the hosted experience? Open [Mogplex in the browser](https://www.mogplex.com/login),
connect GitHub, and start from Projects, Triggers, or Automations.
