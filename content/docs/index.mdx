Mogplex ships in two surfaces.

Use the CLI when you want local, repo-in-place agent work from your terminal.
Use the web app when you want the shared control plane for projects, agents,
automations, observability, and settings.

## Quickstart

Install the CLI, point it at a provider key, and run your first task.

```bash
curl -fsSL https://www.mogplex.com/install.sh | sh
export ANTHROPIC_API_KEY=sk-ant-...
mogplex "summarize this repo"
```

`OPENAI_API_KEY` works too if you prefer OpenAI.

Prefer the hosted experience? Open the [Web app](/web), connect GitHub, and
start from Projects or Automations.
