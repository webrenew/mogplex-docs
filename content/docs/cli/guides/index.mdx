- "Why is the CLI choosing this provider or model?"

- "Where should this default live: config file, env var, profile, or flag?"
  Read [Configuration and Flags](/cli/guides/configuration-and-flags).
- "Should this task be interactive or one-shot?"
  Read [Exec Mode](/cli/guides/exec-mode).
- "Why does this slash command exist in one place but not another?"
  Read [Slash Commands](/cli/guides/slash-commands).
- "How do I make Mogplex behave like part of this repo?"
