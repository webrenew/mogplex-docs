## Verify

```bash
mogplex --version
```

If the binary is on your `PATH`, you should see the installed version string.

## Upgrade

Re-run the installer. It detects an existing installation and replaces the binary in place.

## Uninstall

Delete the `mogplex` binary from the location the installer reported. Your session data
in `~/.mogplex/` is left untouched — remove that directory if you want a clean slate.
