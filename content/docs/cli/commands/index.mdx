title: Commands
description: Reference for every `mogplex` subcommand.
---

Pick a command from the sidebar, or run `mogplex --help` in your terminal to see
the full list.

Each page below covers what the command does and its aliases. For the full flag
list, run `mogplex <command> --help` — that output is the source of truth and
reflects whichever CLI version you have installed.

For interactive workflows that happen inside a live session, use
`mogplex slash list` and the [Quickstart](/cli/quickstart).
